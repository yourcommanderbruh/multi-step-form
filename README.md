# 📋 Multi-Step Form

A multi-step student registration form built with React.js.

## 🔗 Live Demo
👉 https://multi-step-form-prash.netlify.app

## ✨ Features
- 3-step form with smooth navigation
- Real-time validation and error handling
- Data persistence on page refresh (localStorage)
- Progress bar showing current step
- Final review screen before submission
- Fully responsive UI

## 🛠️ Tech Stack
- React.js
- CSS3
- localStorage API

## 🚀 How to Run Locally
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
